# Subvisuwheel

App that runs a game where the user is allowed to spin a wheel for prizes. 

The backend is built with Ruby on Rails while the frontend is built with React.

## Requirements

- Ruby - 2.7.4
- Rails - 7.0.3
- ProgreSQL - 14.3
- React - 18.2.0
- CORS - [insert version]
- asdf - 0.9.0

## How to run

### Backend 
``asdf install``
``bundle install``
``bin/rails server``

### Frontend
``cd frontend/``
``npm install``
``npm start``
